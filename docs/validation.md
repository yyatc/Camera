# Интеграционная валидация на реальной PTZ-камере

## Перед стартом
- Заполнить `config/secrets.env`.
- Проверить `stream.input_rtsp` и `stream.output_rtsp` в `config/config.yaml`.
- Поднять MediaMTX.

## Запуск
- `pip install -r requirements.txt`
- `python -m src.app.main`

## Проверка функционала
1. Человек входит в кадр:
   - появляется белый прямоугольник;
   - камера поворачивается для центрирования;
   - масштаб цели стремится к ~60% кадра.
2. Человек смещается:
   - камера следует за целью по pan/tilt.
3. Человек выходит из кадра:
   - камера переходит в медленный хаотичный поиск.
4. Новый человек найден:
   - поиск прекращается, начинается сопровождение.
5. Оверлей:
   - под bbox справа отображается `ID / total`;
   - справа сверху выводится `Total: чч:мм:сс` и список `ID: N sec`.

## Рекомендуемый первичный тюнинг
- `tracking.detection_confidence`: 0.40-0.55
- `tracking.target_area_ratio`: 0.55-0.65
- `ptz.pan_gain`, `ptz.tilt_gain`: 0.6-0.9
- `ptz.search_pan_speed`: 0.05-0.12
- `ptz.search_step_sec`: 1.0-1.8
